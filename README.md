# React Native

React Native starter app.

## Prerequisites

- [Node] 4.0 or newer
- [Homebrew]
- [Watchman]
- [Flow]

Supported operating systems are Android 4.1 (API 16) and iOS 7.0 or greater.

## Setup

On project folder, run:

    $ npm install

## Run

### iOS Simulator

Open `ios/StarterApp.xcodeproj` and hit run in Xcode.

### Android Simulator

On project folder, run:

    $ react-native run-android
